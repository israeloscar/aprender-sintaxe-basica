## ▶️ Como rodar os exercícios

1. Clone o repositório:
```bash
git clone https://github.com/israeloscar/SintaxeBasica.git
```

2. Abra a pasta no IntelliJ IDEA 💡

3. Navegue até o arquivo `.java` desejado e rode com `Ctrl + Shift + F10`

Ou pelo terminal:
```bash
javac src/Notas.java
java -cp src Notas
```
